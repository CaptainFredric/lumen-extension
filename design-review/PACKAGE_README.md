# Lumen Landing Design Package

This package is for an external design pass on the Lumen landing site.

Use `CLAUDE_DESIGN_PROMPT.md` as the instruction source. The screenshots show the current implementation at desktop and mobile widths. The `docs/` directory is the GitHub Pages source for the live site.

Primary edit targets:

1. `docs/index.html`
2. `docs/styles.css`
3. `docs/script.js`

Do not change the extension mechanics from this package. The landing page should remain accurate to the current product state.
